Servidor de Videojuegos

Este es un simulador de sistema operativo diseñado para gestionar procesos en un entorno de servidor de videojuegos. Implementa mecanismos de planificación Round Robin, gestión de memoria dinámica y sincronización de procesos mediante semáforos. El sistema permite simular diferentes tipos de procesos de un videojuego (jugadores, administradores, partidas) y visualizar su ejecución en tiempo real a través de una interfaz gráfica.

Tecnologías Utilizadas:

Python Tkinter POO Algoritmo de Planificación Round Robin

Hilo Mágico

Este es un E-Commerce dedicado a la venta de productos de tejido artesanal. La plataforma está diseñada para ofrecer una experiencia de compra intuitiva y segura, permitiendo a los usuarios explorar una variedad de productos hechos a mano, agregar artículos a su carrito y completar el proceso de compra de manera eficiente. El sitio web cuenta con un diseño atractivo y funcional, optimizado para diferentes dispositivos, y utiliza tecnologías modernas para garantizar un rendimiento óptimo y una gestión eficaz del inventario y las transacciones.

Tecnologías Utilizadas:

Astro.js Sass FastAPI PostgreSQL
